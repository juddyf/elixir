Dasar Bantuan

  

    Untuk maju melalui permainan, klik kiri- atau tekan spasi atau masukkan kunci. Ketika di menu,      klik kiri untuk membuat pilihan, atau gunakan tombol panah untuk memilih pilihan dan masukkan untuk mengaktifkannya.   

  

Game Menu

  

    Ketika bermain game, klik kanan atau tekan melarikan diri untuk masuk ke menu game. Menu permainan     memberikan pilihan berikut:   

  
    
Kembali
    
Kembali ke permainan.
    
Simpan Game
    
Memungkinkan Anda untuk menyimpan permainan dengan mengklik slot save.
    
Load Game
    
Memungkinkan Anda untuk meload permainan dengan mengklik slot save. Mengklik "Auto" mengakses slot save otomatis.
    
Preferences
    
      Mengubah preferensi permainan (pilihan / konfigurasi):       
        
Tampilan
        
Ganti antara fullscreen dan mode windowed.
        
Transisi
        
Mengontrol tampilan transisi antara layar permainan.
        
Kecepatan Teks
        
Kontrol tingkat di mana menampilkan teks. Lebih lanjut ke kanan slider ini, lebih cepat teks           akan ditampilkan. Semua cara untuk teks penyebab yang tepat yang akan ditampilkan langsung.
        
Joystick
        
Memungkinkan Anda mengontrol permainan menggunakan joystick.
        
Lewati
        
Memilih antara melewatkan pesan yang telah sudah terlihat (dalam bermain apapun melalui permainan), dan           melewatkan semua pesan.
        
Mulai Lompati
        
Kembali ke permainan, sementara melompat.
        
Setelah Pilihan
        
Controls jika melewatkan berhenti setelah mencapai menu.
        
Waktu Auto-Forward
        
Kontrol muka otomatis. Lebih lanjut ke kiri slider ini, yang lebih pendek jumlah waktu           sebelum kemajuan permainan. Semua jalan ke kanan berarti teks tidak akan pernah auto-maju.
        
Music, Volume Suara, dan Voice
        
Mengontrol volume Music, efek suara, dan saluran suara, masing-masing. Lebih lanjut ke           kanan ini, yang lebih keras volume.
      
    
    
Menu Utama
    
Mengembalikan ke menu utama, mengakhiri permainan saat ini.
    
Bantuan
    
Menunjukkan layar bantuan ini.
    
Keluar
    
Keluar permainan; permainan akan ditutup dan berakhir.
  
  

Key dan Mouse Bindings

  
    
Kiri-klik, Enter
    
Kemajuan melalui permainan, mengaktifkan pilihan menu, tombol, dan slider.
    
Ruang
    
Kemajuan melalui permainan, tapi tidak mengaktifkan pilihan.
    
Arrow Keys
    
menavigasi antara menu pilihan, tombol, dan slider.
    
Ctrl
    
Penyebab melewatkan terjadi sementara tombol ctrl ditekan.
    
Tab
    
Mengaktifkan melompat-lompat, menyebabkan ia terjadi sampai tab ditekan lagi.
    
roda mouse-Up, PageUp
    
Penyebab rollback terjadi. Rollback membalikkan permainan kembali waktu, menunjukkan teks sebelum dan bahkan memungkinkan       pilihan menu untuk berubah.
    
roda mouse-Down, PageDown
    
Penyebab rollforward terjadi, membatalkan sebuah rollback sebelumnya.
    
Klik kanan, Escape
    
Memasuki menu game. Ketika di menu permainan, kembali ke permainan.
    
Tengah-klik, H
    
Menyembunyikan jendela teks dan menampilkan sementara lainnya.
    
F
    
Toggles mode fullscreen
    
S
    
Membawa screenshot, menyimpannya dalam sebuah file bernama screenshotxxxx.png, di mana xxxx adalah nomor seri.
    
Alt-M, Command-H
    
Jangat (iconifies) jendela.
    
Alt-F4, Command-Q
    
Quits permainan.
    
Hapus
    
Ketika slot save dipilih, menghapus yang Hemat Slot.
    
v
    
      Matikan mode self-menyuarakan, yang berbunyi teks ke pengguna menggunakan speech synthesizer os-disediakan. Untuk lebih       informasi, silakan baca sendiri menyuarakan       dokumentasi.     
    
Shift + C
    
modus Beralih clipboard-menyuarakan, yang salinan teks ke clipboard sehingga dapat dibaca oleh pembaca layar.
  
  

Kontroler Dukungan dan Bindings

  

    Game ini akan secara otomatis mendeteksi dan pengendali penggunaan permainan didukung oleh     SDL2. kontroler lain dapat dikonfigurasi dengan menggunakan alat konfigurasi pihak ketiga     seperti SDL2 Gampad Alat   

  

    Sejumlah kecil sistem mungkin memiliki masalah dengan menggunakan pengendali permainan terdeteksi. Harus     itu terjadi, tahan shift karena permainan dimulai, dan menonaktifkan dukungan kontroler.   

  

The binding berikut digunakan:

  
    
kanan Trigger, A (Bawah Button)
    
Kemajuan melalui permainan, mengaktifkan pilihan menu, tombol, dan slider.
    
Panduan, Mulai
    
Memasuki menu game. Ketika di menu permainan, kembali ke permainan.
    
Directional Pad, Analog Sticks
    
menavigasi antara menu pilihan, tombol, dan slider.
    
Kiri Trigger, Bahu Kiri, Kembali
    
Penyebab rollback terjadi. Rollback membalikkan permainan kembali waktu, menunjukkan teks sebelum dan bahkan memungkinkan       pilihan menu untuk berubah.
    
Shoulder kanan
    
Penyebab rollforward terjadi, membatalkan sebuah rollback sebelumnya.
    
Y (Top Button)
    
Menyembunyikan jendela teks dan menampilkan sementara lainnya.
